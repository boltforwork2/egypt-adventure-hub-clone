# Quick Folder Creation Commands

## For Windows (Command Prompt)

```batch
cd src\assets

mkdir "Orange Bay Island Images"
mkdir "Paradise Island Images"
mkdir "Dolphin House Images"
mkdir "Scuba Diving Intro Images"
mkdir "Private Speedboat Images"
mkdir "Glass Boat Images"
mkdir "Submarine Trip Images"
mkdir "Quad Bike Safari Images"
mkdir "Super Safari Images"
mkdir "Luxor Trip Images"
mkdir "Cairo Trip Images"
```

## For Windows (PowerShell)

```powershell
cd src/assets

New-Item -ItemType Directory -Name "Orange Bay Island Images"
New-Item -ItemType Directory -Name "Paradise Island Images"
New-Item -ItemType Directory -Name "Dolphin House Images"
New-Item -ItemType Directory -Name "Scuba Diving Intro Images"
New-Item -ItemType Directory -Name "Private Speedboat Images"
New-Item -ItemType Directory -Name "Glass Boat Images"
New-Item -ItemType Directory -Name "Submarine Trip Images"
New-Item -ItemType Directory -Name "Quad Bike Safari Images"
New-Item -ItemType Directory -Name "Super Safari Images"
New-Item -ItemType Directory -Name "Luxor Trip Images"
New-Item -ItemType Directory -Name "Cairo Trip Images"
```

## After Creating Folders

Place exactly 5 images in each folder following the naming pattern from QUICK_REFERENCE.md

Each folder must have:
- [Prefix]Image1.jpg
- [Prefix]Image2.jpg
- [Prefix]Image3.jpg
- [Prefix]Image4.jpg
- [Prefix]Image5.jpg

See QUICK_REFERENCE.md for the exact file names for each folder.
